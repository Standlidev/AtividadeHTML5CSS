# AtividadeHTML5CSS
# Atividade Prática: Website Institucional (HTML5 + CSS3)

## 🎯 Objetivo do Projeto
[cite_start]Este projeto foi desenvolvido como parte de uma atividade prática para consolidar conhecimentos em **HTML5** e **CSS3**. [cite_start]O foco principal foi a estruturação de um site multipáginas funcional, com navegação integrada e estilização customizada[cite: 6, 11, 41].

## 📂 Estrutura de Páginas
[cite_start]O site é composto por 4 páginas principais, todas conectadas entre si através de um menu de navegação[cite: 6, 11]:

* [cite_start]**Home (`index.html`):** Apresenta o título principal, uma seção de notícias com imagens e textos descritivos, além de links para redes sociais[cite: 7, 13, 14, 15, 16, 17].
* [cite_start]**Sobre (`sobre.html`):** Contém a história, missão, visão e valores da empresa, utilizando parágrafos organizados para descrever a área de atuação[cite: 8, 20, 22, 23, 24, 25].
* [cite_start]**Produtos (`produtos.html`):** Catálogo com 5 produtos, incluindo nome, imagem, descrição e valor para cada item[cite: 9, 26, 27, 28, 29, 30, 31, 32].
* [cite_start]**Contato (`contato.html`):** Possui um formulário funcional (Nome, E-mail e Mensagem) e uma representação visual de localização via mapa[cite: 10, 34, 35, 36, 37, 39, 40].

## 🛠️ Especificações Técnicas e Estilização
[cite_start]A folha de estilo (CSS) foi aplicada para garantir a identidade visual do projeto[cite: 41, 42], abrangendo:
* [cite_start]**Cores:** Personalização das cores de fundo e do texto[cite: 43, 44].
* [cite_start]**Tipografia:** Ajuste de fontes e tamanhos para melhor legibilidade[cite: 45, 46].
* [cite_start]**Layout:** Aplicação de bordas e organização de elementos em listas ou tabelas[cite: 33, 47, 48].
* [cite_start]**Organização:** Código legível e links funcionando corretamente conforme as regras gerais[cite: 57, 59].

## 🚀 Como executar
1. Faça o download ou clone este repositório.
2. Certifique-se de que todos os arquivos `.html` e a pasta de imagens estejam no mesmo diretório.
3. Abra o arquivo `index.html` em seu navegador de preferência.
