
# Atividade Prática: Website Institucional (HTML5 + CSS3)

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como parte de uma atividade prática para consolidar conhecimentos em **HTML5** e **CSS3**.O foco principal foi a estruturação de um site multipáginas funcional, com navegação integrada e estilização customizada.

## 📂 Estrutura de Páginas
O site é composto por 4 páginas principais, todas conectadas entre si através de um menu de navegação:

* **Home (`index.html`):** Apresenta o título principal, uma seção de notícias com imagens e textos descritivos, além de links para redes sociais.
* **Sobre (`sobre.html`):** Contém a história, missão, visão e valores da empresa, utilizando parágrafos organizados para descrever a área de atuação.
* **Produtos (`produtos.html`):** Catálogo com 5 produtos, incluindo nome, imagem, descrição e valor para cada item.
* **Contato (`contato.html`):** Possui um formulário funcional (Nome, E-mail e Mensagem) e uma representação visual de localização via mapa.

## 🛠️ Especificações Técnicas e Estilização
A folha de estilo (CSS) foi aplicada para garantir a identidade visual do projeto, abrangendo:
* **Cores:** Personalização das cores de fundo e do texto.
* **Tipografia:** Ajuste de fontes e tamanhos para melhor legibilidade.
* **Layout:** Aplicação de bordas e organização de elementos em listas ou tabelas.
* **Organização:** Código legível e links funcionando corretamente conforme as regras gerais.

## 🚀 Como executar
1. Faça o download ou clone este repositório.
2. Certifique-se de que todos os arquivos `.html` e a pasta de imagens estejam no mesmo diretório.
3. Abra o arquivo `index.html` em seu navegador de preferência.
